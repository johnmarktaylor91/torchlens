## How to add a new visual page

1. Add a row to this table.
2. Add a corresponding tuple to `PAGE_LIST` in `generate_visual_pack.py`:
   `(label, model_key, draw_method, draw_kwargs, caption, extra_setup)`.
3. Run `python notebooks/audit/visual/generate_visual_pack.py` to regenerate.
4. Commit the updated script + this matrix with `docs(audit): add visual page NN ...`.

## Lockstep rule

Whenever a `draw()` kwarg is added, renamed, or removed:
- Update the relevant page row here.
- Update `PAGE_LIST` in `generate_visual_pack.py`.
- Re-run the script; confirm no new GAPs.
