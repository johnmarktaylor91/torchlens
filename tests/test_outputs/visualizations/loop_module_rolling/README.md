# Loop-Module Rolling Visual Demos

These fixtures are regenerated by `tests/test_loop_module_rolling.py` as SVG and PNG files.

## Multi-Op Loop Body

```python
for _ in range(3):
    x = self.block(x)  # Linear -> ReLU
```

Expanded labels: `lin (x3)` and `relu (x3)`.

## Reused Module Inside And Outside A Loop

```python
outside = self.lin(x)
for _ in range(3):
    inside = self.lin(inside)
```

Label: `lin (x4:1,2-4)`.

## Reused Multi-Op Block

```python
for _ in range(3):
    x = self.block(x)
```

Rendered both expanded and collapsed. Expanded op labels use `linear_1_1 (x3)` and
`relu_1_2 (x3)`; the collapsed module box uses `@block (x3)`.

## Buffer Rewrite Loops

```python
for _ in range(2):
    y = y + self.state
    self.state = self.state + 1
for _ in range(3):
    y = y + self.state
    self.state = self.state + 1
```

Buffer address label: `@state:1-6`.

## Two Distinct Loops

```python
for _ in range(2):
    first = self.shared(first)
for _ in range(3):
    second = self.shared(second)
```

Label: `@shared (x5:1-2,3-5)`.

## Label Contract

Rolled nodes keep honest execution counts. Colon ranges are used only when a compact call
or buffer index set disambiguates split regions. Removed forms such as `sites`, `×N`,
facets, `sites?`, and recurrence self-edge glyphs are not part of these demos.
