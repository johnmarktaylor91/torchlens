# TorchLens 2.0 API changes

TorchLens 2.0 reserves exactly 40 top-level names in `torchlens.__all__`. Names
outside that budget either moved to a namespace with a deprecation shim, remain
available only from their canonical namespace, or are new reserved stubs for a
later feature-overhaul phase.

## Wrapper migrations

Forward validation:

```python
# Before
import torchlens as tl
ok = tl.validate_forward_pass(model, x, None, 42)

# After
from torchlens.validation import validate_forward_pass
ok = validate_forward_pass(model, x, None, 42)
```

Backward validation:

```python
# Before
ok = tl.validate_backward_pass(model, x, None, loss_fn)

# After
from torchlens.validation import validate_backward_pass
ok = validate_backward_pass(model, x, None, loss_fn)
```

Saved-activation validation:

```python
# Before
ok = tl.validate_saved_activations(model, x)

# After
from torchlens.validation import validate_saved_activations
ok = validate_saved_activations(model, x)
```

Summary:

```python
# Before
text = tl.summary(model, x)

# After
from torchlens.visualization import summary
text = summary(model, x)
```

Forward graph:

```python
# Before
tl.show_model_graph(model, x, vis_outpath="graph", vis_save_only=True)

# After
from torchlens.visualization import show_model_graph
show_model_graph(model, x, vis_outpath="graph", vis_save_only=True)
```

Backward graph:

```python
# Before
dot = tl.show_backward_graph(model_log, vis_outpath="backward")

# After
from torchlens.visualization import show_backward_graph
dot = show_backward_graph(model_log, vis_outpath="backward")
```

Intervention spec load:

```python
# Before
spec = tl.load_intervention_spec("run.tlspec")

# After
spec = tl.load("run.tlspec")
# or
from torchlens.io import load_intervention_spec
spec = load_intervention_spec("run.tlspec")
```
